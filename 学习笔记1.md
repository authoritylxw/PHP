# 学习笔记1
**PHP基础语法**
PHP 脚本可放置于文档中的任何位置。
```php
<?php
// 此处是 PHP 代码
?>
```
PHP 语句以分号结尾（;）,PHP 代码块的关闭标签也会自动表明分号（因此在 PHP 代码块的最后一行不必使用分号）。

**PHP中的注释**
PHP 代码中的注释不会被作为程序来读取和执行。它唯一的作用是供代码编辑者阅读。

PHP支持三种注释
<li>单行注释 //</li>
<li>单行注释 #</li>
<li>多行注释 /**/</li>

```php
<!DOCTYPE html>
<html>
<body>
<?php
// 这是单行注释

# 这也是单行注释

/*
 这是多行注释块
 它横跨了
 多行
*/
?>
</body>
</html>
```
